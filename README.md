# SkyNest Travel Co. — UI/UX Website Redesign

## Project Overview

This project is a modern travel agency website redesign created as part of the UI/UX Design Internship at Future Interns.

The objective of this project was to design a visually appealing, user-friendly, and conversion-focused travel website that encourages users to explore destinations and book travel packages easily.

---

## Design Goals

* Create a modern and engaging travel website UI
* Improve user experience and navigation flow
* Design clear and effective call-to-action (CTA) sections
* Build a responsive and visually appealing layout
* Enhance user trust through testimonials and branding

---

## Features

* Hero section with booking CTA
* Travel package showcase
* Service information section
* Testimonials / customer feedback section
* Conversion-focused design structure
* Responsive mobile-friendly layout
* Footer with contact information

---

## Wireframes

Wireframes were created to plan:

## Design Decisions

* Gold tones were used to represent luxury and premium travel experiences.
* Large travel imagery was used to create emotional engagement.
* CTA buttons were strategically placed to improve booking conversion.
* Clean layouts and typography were used to improve readability and usability.
